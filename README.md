# AI 图片生成 - 微信小程序

基于微信小程序的 AI 图片生成应用，用户通过输入文字描述生成对应图片，图片生成能力由 Coze API 提供。

## 核心功能（规划中）

- 文字转图片：输入提示词，调用 Coze API 生成图片
- 图片类型选择：人物 / 动漫 / 通用
-  preset 提示词案例
- 保存至相册、重新生成

- Coze API
